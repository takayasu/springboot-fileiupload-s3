# SpringBoot Web Application Sample (Upload And S3)

## Funcation
+ FileUpload
+ AWS S3 PutObject

## Configuration

### AWS
+ ~/.aws/credentials
This configfile config AccessKey and SecretKey.

+ Bucket Name
S3 Bucket Name set at UploadController#BUCKET_NAME constant variable.

### URL
ContextPath /aws
UploadPath upload

Url: http://yourserver/aws/upload
